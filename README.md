# TarlaBazar — прямой рынок от поля до покупателя

Второй сайт проекта. B2B-маркетплейс: фермеры продают свой реальный, уже
собранный урожай напрямую магазинам, ресторанам и переработчикам — без цепочки
перекупщиков, которая поднимает цену в 2–3 раза.

Интерфейс на трёх языках: русский, английский, азербайджанский.
Палитра та же природная, что у AgroLens, — два сайта читаются как одна система.

## Как защищена сделка — пять решений в конструкции

Каждое решение убирает конкретный риск, на котором обычный покупатель или
продавец теряет деньги. Это не правила поверх площадки, а её устройство:

1. Продаётся только реальный, физически имеющийся товар — покупатель не платит
   за то, что существует только на бумаге.
2. Прозрачная комиссия за услугу (2–3%) вместо скрытой наценки перекупщика.
3. Каждый лот с ясными условиями: культура, объём, цена, сорт — до заявки.
4. Только собранный урожай — нельзя выставить то, чего ещё нет на земле.
5. Безопасная оплата через банк-партнёр: деньги как залог до получения товара,
   перевод фермеру только после подтверждения.

За каждым решением стоит реальный провал: международные агро-платформы разорялись
там, где брали на себя чужой риск — займы, скупка товара на склад, предоплата.

## Видео и презентация

Инструкция — в `public/README-положи-сюда-видео-и-презентацию.txt`. Коротко:
- Видео: вставь ID ролика YouTube в `YT_ID` в начале `src/Landing.jsx`.
- Презентация: положи `TarlaBazar.pdf` в `public/`.

## Технически

Рабочий каркас, а не бутафория. Размещённые лоты и заявки реально сохраняются
между сессиями (localStorage). Платформа стартует пустой — никаких выдуманных
фермеров. Пустые состояния честно говорят, что данные появятся с реальными
пользователями.

Экраны: главная, каталог (`/catalog`) с фильтрами, размещение лота (`/new`),
детали лота с формой заявки (`/listing/:id`), заявки (`/deals`) с жизненным
циклом сделки от отправки до подтверждения получения.

## Запуск

Нужен Node.js 18+.

```
npm install
npm run dev      # localhost:5173
npm run build    # в папку dist/
npm run preview
```

## Для боевого запуска (сверх каркаса)

Общая база данных вместо localStorage; интеграция с API банка-партнёра для
реального сплита платежа; верификация фермера через ASAN Login; стыковка с
AgroLens, чтобы сорт качества из фотоанализа подставлялся в лот.

## Технологии

React 18 · Vite · Tailwind CSS · lucide-react · react-router.
