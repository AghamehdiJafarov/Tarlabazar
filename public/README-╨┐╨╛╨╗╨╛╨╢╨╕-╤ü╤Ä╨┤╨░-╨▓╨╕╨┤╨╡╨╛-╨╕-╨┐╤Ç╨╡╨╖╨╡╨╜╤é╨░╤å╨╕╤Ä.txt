Как добавить своё видео и презентацию
======================================

ВИДЕО (промо-ролик на главной странице)
----------------------------------------
1. Загрузи ролик на YouTube (можно "Доступ по ссылке", если не хочешь в поиске).
2. Скопируй ID ролика — часть ссылки после watch?v=
   Пример:  https://www.youtube.com/watch?v=dQw4w9WgXcQ  →  ID = dQw4w9WgXcQ
3. Открой  src/Landing.jsx  в текстовом редакторе.
4. В начале файла найди:   const YT_ID = "REPLACE_WITH_YOUTUBE_ID";
5. Замени на свой ID:       const YT_ID = "dQw4w9WgXcQ";
6. Сохрани. На месте заглушки появится ролик.

Пока ID не заменён — показывается аккуратная заглушка, сайт не ломается.

ПРЕЗЕНТАЦИЯ (PDF по кнопке)
----------------------------
1. Сохрани презентацию в PDF, назови файл  TarlaBazar.pdf
2. Положи его в эту папку (public/), рядом с этим файлом.
3. Кнопки "Скачать презентацию" начнут отдавать этот файл.

Другое имя? Открой src/Landing.jsx И src/ui.jsx, найди в начале каждого:
   const DECK_FILE = "/TarlaBazar.pdf";
и поменяй имя в обоих (слэш в начале оставь).
